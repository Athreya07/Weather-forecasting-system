### First Time Setup

```bat
cd C:\Users\athar\Downloads\Claude\pune-weather-forecasting-system\pune-weather

py -3.12 -m venv venv

venv\Scripts\activate

pip install -r backend\requirements.txt
```

### Start Backend

```bat
set MODEL_PATH=%cd%\ml\models
set DATA_PATH=%cd%\data

cd backend

uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### Start Frontend

Open a new CMD:

```bat
cd C:\Users\athar\Downloads\Claude\pune-weather-forecasting-system\pune-weather\frontend

npm install

npm start
```
